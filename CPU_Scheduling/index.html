CPU Scheduling

CPU scheduling deals with the problem of deciding which of the processes in the ready queue is to be allocated the CPU. The aim of CPU scheduling is to make the system fast, efficient and fair, and to never keep the CPU idle.

FCFS

First Come First Served

FCFS is the simplest scheduling algorithm where the process that requests the CPU first is allocated the CPU first. It is non-preemptive and is implemented with a FIFO queue.

SJF

Shortest Job First

SJF selects the waiting process with the smallest execution time to execute next (non-preemptively).

SRTF

Shortest Remaining Time First

SRTF is the preemptive version of SJF scheduling.

Priority

Priority Scheduling

Priority scheduling is a method of scheduling processes based on priority. Processes with equal priority are executed on FCFS basis. Both preemptive and non-preemptive versions exist.

RR

Round Robin Scheduling

Round Robin Scheduling treats the ready queue as a circular queue. The CPU scheduler goes around the ready queue, allocating the CPU to each process for a fixed time interval called time quantum.

MLQS

Multilevel Queue Scheduling

A multi-level queue scheduling algorithm divides the ready queue into several separate queues based on response-time requirements of the processes. The queues will be assigned different priorities (processes in higher priority queues are executed first) and have their own individual scheduling algorithms.
